## React Webpack Starter

### Fitur

*  Bundle file dengan webpack
*  ES5 code (dengan Babel)
*  Styling dengan CSS atau SCSS/SASS

### Penggunaan

* Kloning proyek ini ke folder anda

```bash
git clone https://github.com/usamahbass/starter-react-webpack.git my-project
```

* Masuk ke folder yang telah dibuat

```bash
cd my-project
```

### Install Package

* jika anda menggunakan yarn

```bash
yarn install
```

* jika anda menggunakan npm

```bash
npm install
```

### Menjalankan bundle

```bash
yarn build
```

atau 

```bash
npm run build
```

### Menjalankan dalam mode Development

jalankan

```bash
yarn start
```

atau

```bash
npm start
```
untuk menginternalisasi dan menjalankan server pengembangan webpack. Arahkan ke http: // localhost: 8080 /. Aplikasi akan secara otomatis memuat ulang jika Anda mengubah salah satu file sumber.
